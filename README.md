# nlp-spam-detection-system

🧠 Repository Description (README Intro)

A machine learning-based spam detection system built using natural language processing techniques.

The project demonstrates end-to-end text classification, including synthetic data generation, feature extraction using bag-of-words, model training, and performance evaluation using multiple metrics.

🚀 Key Features
🧠 Text Classification Pipeline
Binary classification (Spam vs Ham)
Logistic Regression model
End-to-end ML workflow
→ see:
📝 Synthetic Data Generation
Programmatically generated dataset
Balanced classes (spam vs non-spam)
Controlled vocabulary for experimentation
🔍 NLP Feature Engineering
Bag-of-Words representation using CountVectorizer
Converts raw text → numerical features
📊 Model Evaluation

Includes multiple evaluation metrics:

Confusion Matrix
Classification Report (Precision, Recall, F1-score)
ROC-AUC Score
ROC Curve visualization
🧾 Model Interpretability
Extracts most important words influencing predictions
Identifies:
Top spam-indicating keywords
Top ham-indicating keywords
